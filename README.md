# 🎨 BUTUSIC | Illustrations Exhibition Landing Page

A modern editorial-style landing page inspired by a premium illustrations exhibition poster. This project recreates a visually rich magazine-style UI using **HTML5**, **CSS3**, **CSS Grid**, and **Flexbox**, with a strong focus on layout precision, typography, and responsive design principles.
